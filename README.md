## Overview

This project is a visual database tool that translates SQL into Scratch-style drag-and-drop blocks. It is designed to make database querying accessible to non-technical users by eliminating the need to write raw SQL code.

Users can construct queries visually using intuitive blocks that represent core SQL operations such as `SELECT`, `WHERE`, `JOIN`, and aggregation. The system then converts these blocks into valid SQL queries in real time.

---

## Background

This project was developed during a hackathon and was recognized as a **Finalist (Top 6 out of 80 teams)** in South Luzon at the **Byte Forward Hackathon 2025**.

The goal of the project was to bridge the gap between technical and non-technical users by simplifying database interactions through a visual programming interface.

---

## Key Features

* **Drag-and-Drop Query Builder**
  Construct SQL queries using Scratch-inspired visual blocks.

* **Automatic SQL Generation**
  Instantly converts block configurations into valid SQL syntax.

* **Support for Core SQL Operations**

  * `SELECT` (column selection)
  * `WHERE` (filter conditions)
  * `JOIN` (multi-table queries)
  * Aggregations (`COUNT`, `SUM`, `AVG`, etc.)

* **Beginner-Friendly Interface**
  Designed for users with little to no SQL experience.

* **Real-Time Feedback**
  Updates generated SQL dynamically as blocks are modified.

---

## How It Works

1. Users drag blocks representing SQL operations into a workspace.
2. Blocks snap together to form a logical query structure.
3. The system validates the block connections.
4. A corresponding SQL query is generated in real time.
